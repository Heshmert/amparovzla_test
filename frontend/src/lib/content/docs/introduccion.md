# ¿Qué es AMPAROvzla?

AMPARO es una herramienta directa, limpia y vecinal pensada para organizar la ayuda en momentos difíciles. No es una red social para perder el tiempo ni una página corporativa con adornos. Está diseñada para que los centros de acopio y las comunidades se pongan de acuerdo rápido, sepan qué hace falta, dónde se necesita apoyo y quiénes están bien.

## Lo que hace el sistema hoy

- **Censo sencillo:** Saber quién está bien, quién está herido o quién falta. Sin formularios interminables.
- **Control de acopios:** Registrar centros de acopio y llevar el inventario real de lo que tienen disponible.
- **Fichas de despacho:** Crear y descargar etiquetas limpias para pegarlas a las cajas de insumos, asegurando que no se pierdan en el camino.
- **Vacantes de voluntariado:** Espacio para que la gente se postule a ayudar según lo que sabe hacer.
- **Reportes y denuncias:** Un buzón abierto para que los vecinos dejen alertas directas de lo que ocurre en sus calles.

## Nuestra filosofía de diseño
Buscamos que la página cargue al tiro, consuma el mínimo de datos de internet en el teléfono y sea fácil de entender para cualquiera. Por eso evitamos el desorden visual, las tarjetas gigantes de colores y las animaciones pesadas. El orden y la sobriedad calman; el desorden estresa.
