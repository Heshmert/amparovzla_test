# Cómo se ve y se siente AMPARO

La interfaz de la plataforma se divide en dos enfoques visuales claros, pero ambos comparten la misma geometría rígida: **bordes completamente rectos (cero esquinas redondeadas)** y líneas finas para separar las secciones.

## 1. El Panel de Gestión (Estilo "Stone")
Es el espacio de trabajo para los centros de acopio, formularios internos y el control de inventario. Está pensado para ser denso, funcional y enfocado puramente en los datos.
- **Fondo:** Color arena claro (`bg-stone-50`).
- **Contenedores:** Blanco puro (`bg-white`).
- **Texto y botones principales:** Gris oscuro industrial (`text-stone-900`).
- **Tablas:** Compactas, pegadas (`border-collapse`) y con letras chiquitas en mayúsculas (`text-[10px] font-black tracking-clean`) para que quepa mucha información en una sola pantalla.

## 2. Páginas Públicas (Estilo "Bandera + Alto Impacto")
Es lo que ve el público general cuando entra a buscar información, leer las guías de talentos o revisar las novedades. Aquí usamos la fuerza visual de la bandera para captar la atención:
- **Títulos masivos:** Letras extra negras (`font-black`) y tamaños gigantes que se adaptan a la pantalla.
- **Colores de acento:** 
  - **Amarillo/Amber:** Para llamadas de atención o estados pendientes.
  - **Azul (`blue-700`):** Para las acciones principales y bloques de comunidad.
  - **Rojo (`red-700`):** Para las alertas críticas, errores o emergencias.
